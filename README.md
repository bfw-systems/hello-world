# bfw-hello-world
Hello World module for BFW

This is a module to introduce the module system in BFW v3.x

The second utility is to test module installation system.

# Installation
We recommend to use composer for installation.

Download composer : https://getcomposer.org/download/

Run command : `composer require bulton-fr/bfw-hello-world`

And run `./vendor/bin/bfw_installModules`
